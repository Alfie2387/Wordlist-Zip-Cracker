# Wordlist-Zip-Cracker
A programme to crack the passwords on zip files using a wordlist, eg rockyou 

Steps:
1. Download and open the python file
2. Enter the file path where prompted
3. Enter the path for the word list you wish to use eg rockyou
You can download the rockyou word list here:
https://github.com/josuamarcelc/common-password-list
